Welcome!

JSLint seems to be proud of the fact that it will "hurt your feelings". Well that's OK, but it would be nice if it made it a bit clearer why it was hurting them.

JSLint Error Explanations is designed to help you improve your JavaScript by understanding the sometimes cryptic error messages produced by JSLint, and teaching you how to avoid such errors.

This site is still a work in progress. Not all JSLint error messages are documented here yet. The code is all available on GitHub if you would like to help out! Alternatively, you can request an article for a specific error message by contacting me or raising an issue on GitHub.
